# LIAMSTER69-FPS

A small multiplayer FPS with wacky weapons like egg launchers and banana RPGs.

## How to run

### Server (Replit / Local / Render)
1. Navigate to `server/` folder
2. Install dependencies: `npm install`
3. Start server: `npm start`
4. Copy the URL (Replit example: `wss://your-repl-url`) to the client

### Client
1. Open `client/index.html` in browser
2. In `client/main.js`, replace `YOUR-REPLIT-OR-SERVER-URL` with your server URL

### Notes
- Designed for small friend groups
- Free hosting works (Replit / Glitch)
- Expandable with weapons and more wacky features
